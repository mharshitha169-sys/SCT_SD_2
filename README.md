# 🎯 Number Guessing Game

## 📖 Description
This project is a simple console-based **Number Guessing Game** developed in Java as part of my **Software Development Internship** at **SkillCraft Technology**. The application generates a random number within a specified range, and the user attempts to guess it. The program provides feedback after each guess, indicating whether the entered number is too high or too low, until the correct number is guessed.

## ✨ Features
- Random number generation
- User-friendly console interaction
- Hints for incorrect guesses (Too High / Too Low)
- Tracks the number of attempts
- Displays a success message upon guessing the correct number

## 🛠️ Technologies Used
- Java
- Object-Oriented Programming (OOP)
- Scanner Class
- Random Class

## 🚀 Getting Started

### Prerequisites
- Java JDK 8 or later
- Any Java IDE (VS Code, IntelliJ IDEA, Eclipse) or Command Prompt

### Steps to Run
1. Clone the repository.
2. Open the project in your preferred Java IDE.
3. Compile and run the Java source file.
4. Enter your guesses and follow the hints until you find the correct number.
